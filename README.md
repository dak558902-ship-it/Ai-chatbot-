# AI Chatbot Using Python, NLP & Deep Learning

## Project Overview

This project is a simple AI Chatbot built using:

* Python
* Natural Language Processing (NLP)
* TensorFlow / Keras
* Deep Learning
* Flask

The chatbot can understand user messages, classify intents, and generate appropriate responses.

## Features

* Intent Recognition
* Deep Learning Model
* NLP Text Processing
* Interactive Chat Interface
* Easy to Customize

## Technologies Used

* Python
* TensorFlow
* Keras
* NLTK
* Flask
* NumPy

## Installation

```bash
git clone https://github.com/yourusername/AI-Chatbot.git

cd AI-Chatbot

pip install -r requirements.txt
```

## Train the Model

```bash
python train.py
```

## Run Chatbot

```bash
python chatbot.py
```

## Run Web Application

```bash
python app.py
```

Open:

http://127.0.0.1:5000
